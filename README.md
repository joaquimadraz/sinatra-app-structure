Sample App
=====================

#### What is inside?

Just a boilerplate I use for my personal projects.

#### Requirements:

* rename `remove_prefix_config.yml` to `config.yml`
* add your `TWITTER_API_KEY`
* add your `TWITTER_API_SECRET`

#### To run
* 1. `bundle install`
* 2. `rake db:create`
* 3. `rake db:migrate`
* 3. `rackup -p 5000`
